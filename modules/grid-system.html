Grid system

Our grid system is based on flexbox. It is not well suited for manual coding, but made in respect of code generation.

Grid system supports any count of columns between 2 and 16. If you project require more or less count of possible grid systems, modify $as-grid-max-cols variable.

Breaking the grid

Responsive design dictate that layout must change to respect device capabilities. Other grid systems allows to define cell width in depends on device mode. However, most of real life cases use it to break row with big number of elements into sub-rows.

This demo shows how this approach works. Click on buttons to change local device mode.
