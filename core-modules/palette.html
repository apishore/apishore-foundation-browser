Material palette code module

This core module defines set of SCSS variables designed to handle material palettes.

https://material.google.com/style/color.html

Named palettes

Most of framework classes do not use palettes directly. Instead of that use configurable aliases. Background and foreground
  • $as-palette-bg: $as-palette-white;
  • $as-palette-fg: $as-palette-black;
Primary ui color
  • $as-palette-primary: $as-palette-red;
  • $as-palette-primary-contrast: $as-palette-bg;
Accent ui color
  • $as-palette-accent: $as-palette-blue;
  • $as-palette-accent-contrast: $as-palette-bg;
Messaging
  • $as-palette-success: $as-palette-green;
  • $as-palette-success-contrast: $as-palette-bg;
  • $as-palette-warn: $as-palette-pink;
  • $as-palette-warn-contrast: $as-palette-bg;
  • $as-palette-error: $as-palette-red;
  • $as-palette-error-contrast: $as-palette-bg;
Branding
  • $as-palette-brand: $as-palette-blue;
  • $as-palette-brand-contrast: $as-palette-fg;
  • $as-palette-sidebar: $as-palette-blue-grey;
  • $as-palette-sidebar-contrast: $as-palette-fg;
Usage color: map-get($as-palette-grey, 500);

Named colors

All material colors accompanied by handy classes
  • .as-color-{palette}-{variant}
  • .as-bg-color-{palette}-{variant}
Also there are two generic colors for each palette
  • .as-color-{palette} : map-get($map, 500);
  • .as-bgcolor-{palette}: map-get($map, 700);
