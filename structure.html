Framework structure

Framework consist of core modules and component modules. You can disable or tune some of them via configuration variables.

  • init.scss
  • style.scss
// Import initial variables like color palette, base units like $gutter, etc. @import "{path-to-foundation}/init.scss" // override default variables, configure grids and used modules $as-palette-primary: $as-palette-blue; $as-palette-primary-contrast: $as-palette-bg; @import "{path-to-foundation}/style.scss" // your css
